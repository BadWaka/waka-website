# waka-website
我的网站

基于node v7.6.0 以上

# 依赖库

### 依赖

- cheerio 爬虫HTML解析
- koa Node框架
- koa-mount 将中间件挂载到特定url下
- koa-router koa路由
- koa-static koa静态资源
- request 简化网络请求
- request-promise 网络请求默认变为promise，方便async/await使用
- swig swig模板引擎
- tracer console.log加强
- ws websocket实现库

### 开发依赖

- gulp 自动化构建工具
- gulp-sass 编译sass
